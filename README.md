# 💻 Web Development Projects

This repository contains a collection of my frontend projects built using **HTML, CSS, and JavaScript**.
Each project focuses on strengthening core concepts like DOM manipulation, event handling, and local storage.

## 🚀 Projects

A simple and interactive task manager.

**Features:**

- Add, edit, delete tasks
- Mark tasks as completed
- Clear all tasks
- Data stored in localStorage

📂 Folder: `todo-app`

A tool to calculate Body Mass Index based on height and weight.

- Displays health category

### ⭐ 4. Review Collector

A simple app to collect and display user feedback.
